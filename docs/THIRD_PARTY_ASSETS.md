# Third-party assets

## Pet Cats Pack

- Asset: Pet Cats Pack
- Used item: Cat-2 Run animation (`Cat-2/Cat-2-Run.png`)
- License: Creative Commons Zero (CC0)
- Source: [Pet Cats Pack](https://luizmelo.itch.io/pet-cat-pack)
- Modifications: The original 400×50 sprite strip was split into eight 50×50 Dashboard runtime frames. The pixel content of those Dashboard frames was not otherwise modified. Tray animation icons are derived offline from the same eight frames by alpha-bound cropping and nearest-neighbor scaling into transparent 16×16, 20×20, 24×24, and 32×32 icon entries.
- The original `License.txt` states that the pack may be used in commercial and non-commercial projects.

The complete asset pack is not copied into the repository. The root `assets/` directory is an ignored local import source; only the eight imported Dashboard runtime frames, the derived tray animation icons, and the existing static tray fallback are tracked under `src/RunCatDashboard.App/Assets/RunCat/`.
