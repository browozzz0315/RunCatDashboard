# Third-party assets

## Pet Cats Pack

- Asset: Pet Cats Pack
- Used item: Cat-2 Run animation (`Cat-2/Cat-2-Run.png`)
- License: Creative Commons Zero (CC0)
- Source: [Pet Cats Pack](https://luizmelo.itch.io/pet-cat-pack)
- Modifications: The original 400×50 sprite strip was split into eight 50×50 runtime frames. The pixel content was not otherwise modified.
- The original `License.txt` states that the pack may be used in commercial and non-commercial projects.

The complete asset pack is not copied into the repository. The root `assets/` directory is an ignored local import source; only the eight imported runtime frames are tracked under `src/RunCatDashboard.App/Assets/RunCat/`.
